## 课程目标

了解前端开发的主要流程和主流技术，学习前端开发的基本知识。

## 课程内容

* 了解前端开发主要工作和成长路径。
* 安装前端开发工具并配置开发环境。
* 掌握 html、css和javascript网页开发语言。
* 掌握WebAPI的接口和操作方法。
* 掌握jQuery库。
