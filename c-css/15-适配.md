# 移动端适配

## 视口

视口表示浏览器中显示文档的区域。

css 可以设置视口相对单位：

1. 1 vw = 1 / 100 视口宽度
2. 1 vh = 1 /100 视口高度

可以使实现不同宽度的设备中，网页元素尺寸等比缩放效果。

```html
<style>
  * {
    margin: 0;
    padding: 0;
  }

  .box {
    margin: auto;
    width: 300px;
    height: 300px;
    border: 1px solid black;
  }

  .sub {
    width: 10vw;
    height: 10vh;
    background-color: skyblue;
  }
</style>

<div class="box">
  <div class="sub"></div>
</div>
```
