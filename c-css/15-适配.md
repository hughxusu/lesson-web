# 移动端适配

移动端适配是指让网页或应用程序在各种移动设备（如智能手机、平板电脑等）上能够完美显示和正常使用的技术和策略。

* 原因：移动设备具有不同的屏幕尺寸、分辨率、像素密度等特性。
* 解决方案：`rem`单位或`vw/vh`单位

## `rem`单位



## 视口

视口表示浏览器中显示文档的区域。

css 可以设置视口相对单位：

1. 1 vw = 1 / 100 视口宽度
2. 1 vh = 1 /100 视口高度

可以使实现不同宽度的设备中，网页元素尺寸等比缩放效果。

```html
<style>
  * {
    margin: 0;
    padding: 0;
  }

  .box {
    margin: auto;
    width: 300px;
    height: 300px;
    border: 1px solid black;
  }

  .sub {
    width: 10vw;
    height: 10vh;
    background-color: skyblue;
  }
</style>

<div class="box">
  <div class="sub"></div>
</div>
```
